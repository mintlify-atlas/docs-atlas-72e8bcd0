SansiStore is a full-stack e-commerce platform built for Universidad Mayor de San Simón (UMSS), Bolivia. It covers the entire order lifecycle — from customers browsing the product catalog to sellers processing orders, couriers completing deliveries, and administrators reconciling payments. The platform runs on Astro with React, Firebase, and Tailwind CSS, and is deployed to Vercel as a Progressive Web App.

## Get Running Quickly

<Steps>
  <Step title="Install dependencies">
    Install [Bun](https://bun.sh) and Java 21, then run `bun install` inside the repo.
  </Step>
  <Step title="Configure environment variables">
    Copy `.env.example` to `.env` and fill in your Firebase project credentials (or use the emulator defaults for local development).
  </Step>
  <Step title="Start the Firebase emulators">
    Run `bun run emu` to start Firestore, Auth, Functions, and Pub/Sub emulators locally.
  </Step>
  <Step title="Seed and run">
    Seed test data with `bun run seed`, then start the dev server with `bun dev`. Open [http://localhost:4321](http://localhost:4321).
  </Step>
</Steps>

<Note>
  Authentication is restricted to `@umss.edu` Google accounts in production. In development with emulators, any credential works. See [Environment Setup](/environment-setup) for full details.
</Note>
